Tabla: Alerta
CampoTipo de DatoLong.DescripciónRestricciones
id_alertaINT—Identificador único de cada notificación generada por el sistema.PK, NOT NULL, AUTO_INCREMENT
id_productoINT—Producto que originó la alerta (stock bajo o próximo a vencer).FK → Producto(id_producto), NOT NULL
tipo_alertaENUM—Categoría: 'stock_bajo' (stock ≤ mínimo) o 'vencimiento' (≤ 30 días).NOT NULL
mensajeTEXT—Texto informativo que describe el problema para orientar al administrador.NOT NULL
fecha_alertaDATETIME—Fecha y hora en que el sistema generó la notificación.NOT NULL, DEFAULT NOW()
leidaTINYINT1Estado: 0 = pendiente, 1 = leída y resuelta por el administrador.NOT NULL, DEFAULT 0
