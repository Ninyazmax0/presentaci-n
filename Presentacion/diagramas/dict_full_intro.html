Diccionario de Datos
El presente Diccionario de Datos describe de forma detallada cada una de las tablas y sus campos que conforman la base de datos del sistema de inventario de la Farmacia Santa Cruz. Para cada campo se especifica: nombre, tipo de dato, longitud, descripción funcional y restricciones aplicadas.
Leyenda de Restricciones
PK  Llave Primaria
FK → Tabla(campo)  Llave Foránea
NOT NULL  Obligatorio
AUTO_INCREMENT  Auto generado
UNIQUE  Valor único
9 tablas conforman el sistema: Usuario, Proveedor, Categoria, Producto, Compra, DetalleCompra, Venta, DetalleVenta, Alerta.
